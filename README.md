# Steps to install and run 
Ensure Python 3 is installed on your system
Save the code as plagiarism_checker.py
Prepare two text files you want to compare in the same directory
Open terminal/command prompt in the project directory
Run the command: python plagiarism_checker.py
Enter the first filename when prompted (e.g., file1.txt)
Enter the second filename when prompted (e.g., file2.txt)
View the similarity results and feedback
# Instructions for testing
